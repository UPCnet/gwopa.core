==========
gwopa.core
==========

In this package is implemented the funcionality to GWOPA PMP

Features
--------

- Create new project
- Planning
- Monitoring
- Dashboard
- Reporting
- Settings for configuration: Users, Working Areas, WOP Platforms, WOP Programmes, Water and/or Sanitation Operator, Donors ...
